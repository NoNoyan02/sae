Mission 1 : Étude des données

Dans cette première mission, nous avons étudié les données de l'entreprise KDou pour en comprendre la structure et les caractéristiques.

Bilan de l’étude des données

Nous avons analysé les données de KDou en nous concentrant sur les éléments principaux suivants :

  • Structure des tables de données
  • Relations entre les entités (ex : clients, produits)
  • Qualité des données (ex : données manquantes ou incohérentes)

Exemple d’analyse :

Entité Description Observations
Clients Données des clients réguliers et ponctuels La plupart des clients sont réguliers, mais il existe des valeurs manquantes pour certains champs.
Produits Catalogue de produits, incluant desserts et boissons Une majorité des produits sont des desserts, et certaines informations de provenance sont manquantes.

Perspectives d’amélioration et Conclusion

Pour améliorer la qualité de la base de données, nous recommandons de :

  • Ajouter des contrôles pour éviter les données manquantes.
  • Standardiser les formats de données, notamment pour la provenance des produits.

En conclusion, cette mission nous a permis de mieux comprendre les données disponibles chez KDou et de poser les bases d’une exploitation optimale.
