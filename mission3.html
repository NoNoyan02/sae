Mission 3 : Analyse SQL

Dans cette mission, nous avons analysé les données de KDou à l'aide de requêtes SQL sur plusieurs thèmes.

Thème : Desserts / Boissons

Quelles sont les fournisseurs qui vendent uniquement des boissons ou des desserts :

Code SQL : SELECT DISTINCT Fournisseur.NoFour, Fournisseur.Societe FROM Fournisseur WHERE Fournisseur.NoFour IN ( SELECT Pi.NoFour FROM Produit AS P1 INNER JOIN Categorie AS C1 ON P1.CodeCateg = C1.CodeCateg WHERE C1.NomCateg = Desserts JAND Fournisseur,NoFour IN [ SELECT P2.NoFour FROM Produit AS P2 INNER JOIN Categorie AS C2 ON P2.CodeCateg- C2.CodeCateg WHERE C2,NomCateg = "Boissons" );
